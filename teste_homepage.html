Análise de Séries Temporais

Plataforma para análise de séries temporais de dados empregando métodos e modelos consagrados. Análise do comportamento de séries e predição com base nos dados.

Plataforma desenvolvida no Instituto Militar de Engenharia (IME)

Processar »

Open Source

A plataforma possui código aberto, permitindo que o usuário possa verificar e contribuir com os métodos disponibilizados. Além disso a plataforma é 100% gratuita, tendo sido desenvolvida em ambiente acadêmico.

Python

Todos os métodos disponibilizados foram implementados empregando a liguagem de programação Python 3. Assim, caso o usuário deseje obter os códigos de cada um dos métodos eles estarão disponíveis e poderão ser alterados para fins de projetos mais específicos.

Análises Geoespaciais

Desenvolvida na Seção de Engenharia Cartográfica do Instituto Militar de Engenharia, a plataforma permite a análise de séries temporais de dados espaciais.
